Running JavaScript

  • You can run JS on the browser devTools console.

  • In your html file create script tags, inside the script tags write the code you want to run.

    Note: The script tags should always be put before the closing body tag, the reason for this is that you will most likely be using elements of your html in your script, and if the script runs before the elements get created the website will crash.

  • Create a .js file and give it's path to src="" in the script tag, the code written in the .js file will run when the html is loaded.

  • using node: in the console type node followed by the .js filename and this will run the JavaScript code in the file.
